## Leetcode Visualizer

### Features

- Light/Dark Mode
- Difficulty wise plots (Easy, Medium, Hard)
- Category wise plots (Array, Backtracking etc...)
- Entirely lives in user's browser
  - This also implies, no data is being collected by anyone.
